Astro is a modern web framework that lets you build fast, content-focused websites. In this post, we'll walk through how to get started.

## Why Astro?

Astro ships zero JavaScript by default, resulting in faster page loads and better performance scores.

## Installation

```bash
npm create astro@latest
```

Follow the prompts to set up your project.

## Project Structure

```
src/
├── components/
├── layouts/
├── pages/
└── content/
```

Happy coding!
